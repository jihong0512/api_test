## 时机5：执行接口测试任务时（❌ 当前代码不支持自动生成）

**代码位置**：`backend/app/celery_tasks_execution.py` 第200-350行（`execute_interface_task_task` 函数）

**当前逻辑**：
```python
def execute_interface_task_task(self, task_id: int):
    """执行接口测试用例任务"""
    db = SessionLocal()
    try:
        task = db.query(TestTask).filter(TestTask.id == task_id).first()
        if not task:
            raise Exception(f"测试任务不存在: {task_id}")
        
        # 获取用例ID列表
        test_case_ids = json.loads(task.test_case_ids) if task.test_case_ids else []
        
        if not test_case_ids:
            raise Exception("测试用例列表为空")  # ❌ 如果为空，直接抛出异常
```

**结论**：
- ❌ **当前代码不支持自动生成**：如果 `test_case_ids` 为空，会直接抛出异常
- ❌ 不会在执行任务时自动生成测试用例
