RAG检索：
```

    )
    assert resp.status_code == 200
```

**实际例子**（3个关联用例）：

**Step 1: 登录**
```python
# 请求
POST /api/auth/login
{"username": "user1", "password": "pwd123"}

# 响应
{"code": 0, "data": {"token": "abc123xyz", "userId": 42}}

# 提取
extracted_data = {"token": "abc123xyz", "userId": 42}
```

**Step 2: 创建订单**
```python
# 请求配置（包含占位符）
POST /api/orders
headers: {"Authorization": "Bearer {token}"}
json: {"userId": "{userId}", "total": 99.9}

# 自动替换后
headers: {"Authorization": "Bearer abc123xyz"}
json: {"userId": 42, "total": 99.9}

# 响应
{"code": 0, "data": {"orderId": "order_456"}}

# 提取
extracted_data.update({"orderId": "order_456"})
```

**Step 3: 查询订单**
```python
# 请求配置
GET /api/orders/{orderId}
headers: {"Authorization": "Bearer {token}"}

# 自动替换后
GET /api/orders/order_456
headers: {"Authorization": "Bearer abc123xyz"}

# 响应
{"code": 0, "data": {"orderId": "order_456", "status": "completed"}}
```

**完整执行流程**：
```
初始化 extracted_data = {}
  ↓
Step 1: 登录 ✅
  - 提取: token, userId
  - extracted_data = {token, userId}
  ↓
Step 2: 创建订单 ✅
  - 占位符替换: {token}, {userId}
  - 提取: orderId
  - extracted_data = {token, userId, orderId}
  ↓
Step 3: 查询订单 ✅
  - 占位符替换: {token}, {orderId}
  - 提取成功
  ↓
生成报告
  - 3个用例全部通过
  - 自动流转了5个数据
```
