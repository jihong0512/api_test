# APITest 本地开发模式部署总结

## 📋 部署架构

**Docker 容器运行**（基础服务）：
- MySQL (端口: 3309)
- Redis (端口: 6382)
- Neo4j (端口: 7474/7687)
- MinIO (端口: 9005/9006)

**本地运行**（应用服务）：
- Backend (端口: 8004)
- Frontend (端口: 3006)
- Celery Worker

## 🚀 快速启动步骤

### 1. 启动基础服务（Docker）

```bash
cd /Users/jihongqing/new_ai_workspace/apitest
./start_local_dev.sh
```

或者：

```bash
docker-compose -f docker-compose.dev.yml up -d
```

### 2. 配置后端环境变量

```bash
cd backend
cp .env.example .env
```

**重要**：确保 `.env` 文件中的端口配置正确：
- `MYSQL_PORT=3309`（不是3306）
- `REDIS_PORT=6382`（不是6379）
- `MINIO_ENDPOINT=localhost:9005`（不是9000）

### 3. 安装依赖并启动服务

**Backend:**
```bash
cd backend
python3 -m venv venv  # 创建虚拟环境（首次）
source venv/bin/activate
pip install -r requirements.txt
python start_server.py
```

**Celery Worker（新终端）:**
```bash
cd backend
source venv/bin/activate
celery -A app.celery_app worker --loglevel=info --concurrency=4
```

**Frontend（新终端）:**
```bash
cd frontend
npm install  # 首次安装
npm start
```

## 🔧 关键配置说明

### 端口映射（必须匹配）

基础服务端口映射到本地，应用通过 localhost 访问：

| 服务 | Docker内部端口 | 本地端口 | 配置项 |
|------|---------------|---------|--------|
| MySQL | 3306 | **3309** | `MYSQL_PORT=3309` |
| Redis | 6379 | **6382** | `REDIS_PORT=6382` |
| Neo4j Bolt | 7687 | **7687** | `NEO4J_URI=bolt://localhost:7687` |
| MinIO API | 9000 | **9005** | `MINIO_ENDPOINT=localhost:9005` |

### MinIO 配置

MinIO 访问凭据（与 docker-compose.dev.yml 一致）：
- Access Key: `minioadmin`
- Secret Key: `minioadmin123456`

## ⚠️ 常见问题

### 端口被占用
```bash
# 检查端口占用
lsof -i :3309
lsof -i :6382
lsof -i :8004
lsof -i :3006
```

### 无法连接数据库
- 确认 Docker 容器运行：`docker-compose -f docker-compose.dev.yml ps`
- 检查 `.env` 文件端口配置是否正确
- 查看容器日志：`docker-compose -f docker-compose.dev.yml logs mysql`

### Celery Worker 无法启动
- 确认 Redis 服务运行：`docker-compose -f docker-compose.dev.yml ps redis`
- 检查 Redis 连接配置
- 查看 Celery 日志确认错误

## 🔄 停止服务

**停止基础服务：**
```bash
docker-compose -f docker-compose.dev.yml stop
# 或完全停止（保留数据）
docker-compose -f docker-compose.dev.yml down
```

**停止本地服务：**
在各终端按 `Ctrl+C` 停止 Backend、Celery Worker、Frontend
